# Biceps_e-commerce_app
A Gym e-commerce app using MERN stack with context api.

This is an ecommerce website for gym and fitness products using MERN stack.
It has features like order listing, payment integration , cart and wishlist functionalities, category and price filters, searching features. 
User dashboard for viewing orders and update profile
Admin have ability to manage products like - create , update and delete products. And manage categories like - create , update and delete category, view all products and orders , handle orders
Used context api for global state management.
Followed by MVC architecture, front-end as html, css, bootstrap and react. Js in back-end REST api created using node. Js, express. Js and database as Mongo db. Deployed using cyclic.sh
 In back-end ,REST api’s created using node. Js, express. Js and database using Mongo db.
